# Keylogger
Ferramenta que captura cada dígito do teclado da "vítima" e salva num arquivo de log.

## AVISO LEGAL
Se você essa ferramenta em dispositivos de outras pessoas sem autorização é crime. Use apenas na sua própria máquina.

## Sobre
programa ou dispositivo projetado para monitorar e registrar tudo o que você digita no teclado do seu computador ou celular. Imagine-o como um "gravador invisível": cada tecla que você pressiona, seja uma senha, um número de cartão de crédito ou uma mensagem privada é anotada e enviada para quem instalou a ferramenta.

## Como usar
```
python3 keylogger.py
```
Rode o script e digite normalmente. Tudo que for digitado será salvo no arquivo `keylog.txt`. Para encerrar, pressione `Ctrl+C`.

## O que aprendi
1 - Como o teclado é monitorado a nível de sistema
2 - Como teclas especiais (backspace, ctrl, shift) são tratadas diferentes de teclas comuns
3 - Aprendi também como salvar dados em arquivos em tempo real 'flush()'
